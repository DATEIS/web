### Local Tests

To start the local tests, you need to run these commands:

First of all, you need to switch your branch: 
(frontend)

1. git checkout -b <name_of_your_page>-page

Next, you need to run this command:

2. pre-commit run --all-files

If you've got the success result, you can send your work to PR

3. git add .
4. git commit -m "Your message"
5. git push origin <your_branch>
